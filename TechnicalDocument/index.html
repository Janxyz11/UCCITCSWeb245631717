General

Use these general shortcuts to quickly navigate and manage VS Code.

Ctrl + Shift + P, F1 Show Command Palette

Ctrl + P Quick Open, Go to File…

Ctrl + Shift + N New window/instance

  • Ctrl + Shift + W Close window/instance
  • Ctrl + , User Settings
  • Ctrl + K Ctrl + S Keyboard Shortcuts
Basic Editing

These shortcuts help you with basic text editing tasks.

Ctrl + X Cut line (empty selection)

Ctrl + C Copy line (empty selection)

  • Alt + ↑ / Alt + ↓ Move line up/down
  • Shift + Alt + ↓ / Shift + Alt + ↑ Copy line up/down
  • Ctrl + Shift + K Delete line
Navigation

Use these shortcuts to navigate through your code efficiently.

Ctrl + T Show all Symbols

Ctrl + G Go to Line…

  • Ctrl + Shift + O Go to Symbol…
  • Ctrl + - Go back
  • Ctrl + Shift + - Go forward
Search and Replace

These shortcuts help you search and replace text in your files.

Ctrl + F Find

Ctrl + H Replace

  • F3 / Shift + F3 Find next/previous
  • Alt + Enter Select all occurrences of Find match
  • Ctrl + D Add selection to next Find match
Multi-cursor and Selection

These shortcuts help you work with multiple cursors and selections.

Alt + Click Insert cursor

Ctrl + Alt + ↑ / Ctrl + Alt + ↓ Insert cursor above/below

  • Ctrl + U Undo last cursor operation
  • Shift + Alt + I Insert cursor at end of each line selected
  • Ctrl + L Select current line
Rich Languages Editing

These shortcuts enhance your coding experience with language features.

Ctrl + Space Trigger suggestion

Ctrl + Shift + Space Trigger parameter hints

  • Ctrl + . Quick Fix
  • Shift + Alt + F Format document
  • F12 Go to Definition
Editor Management

These shortcuts help you manage your editor windows and tabs.

Ctrl + W Close editor

Ctrl + \ Split editor

  • Ctrl + 1 / Ctrl + 2 / Ctrl + 3 Focus into 1st, 2nd, or 3rd editor group
  • Ctrl + K Ctrl + ← / Ctrl + K Ctrl + → Move editor left/right
  • Ctrl + Tab Open next editor
File Management

These shortcuts help you manage your files within VS Code.

Ctrl + N New File

Ctrl + S Save

  • Ctrl + Shift + S Save As…
  • Ctrl + K Ctrl + S Save All
  • Ctrl + O Open File…
Display

These shortcuts help you control the display and layout of VS Code.

Ctrl + B Toggle Sidebar visibility

Ctrl + J Toggle Panel visibility

  • Ctrl + = / Ctrl + - Zoom in/out
  • Ctrl + 0 Reset zoom level
  • F11 Toggle Full Screen
Debug

These shortcuts help you with debugging your code.

F5 Start/Continue Debugging

F9 Toggle Breakpoint

  • F10 Step Over
  • F11 Step Into
  • Shift + F11 Step Out
Integrated Terminal

These shortcuts help you work with the integrated terminal in VS Code.

Ctrl + ` Toggle Integrated Terminal

Ctrl + Shift + ` Create New Terminal

  • Ctrl + C Copy selection in terminal
  • Ctrl + V Paste into terminal
  • Ctrl + ↑ / Ctrl + ↓ Scroll up/down in terminal
